Exploring List

3 Types of List are here:

Order List:
Order List has Different Types of attribute
  1. type
  2. start
Unordered List:
Unordered List has different types of Style
  • disc - default
  • circle
  • square
  • none
Description List:
Description List contain two tags
  • dt tag for define the description term
  • dd tag for describe the term in a description list
